# 🖥️ Server Stats Script

## 📌 Project URL

https://github.com/Pankaj-Bashera/Performance_Stats

---

## 📖 Overview

This project is a Bash script (`server-stats.sh`) to analyze basic server performance metrics on a Linux system.

It provides:

* CPU usage
* Memory usage (used vs free with %)
* Disk usage (used vs free with %)
* Top 5 processes by CPU usage
* Top 5 processes by memory usage

---

## 🚀 Steps to Execute

### 1. Clone the repository

```bash
git clone https://github.com/Pankaj-Bashera/Performance_Stats.git
cd Performance_Stats
```

### 2. Give permission to script

```bash
chmod +x server-stats.sh
```

### 3. Run the script

```bash
./server-stats.sh
```
